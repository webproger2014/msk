Использование API

Адрес для использования api: "?controller=api/concert"
Метод перадаётся также GET параметром - action
Пример: "action=recordUserConcert"
Пример использования запроса: "?controller=api/concert&action=recordUserConcert"

1 Метод регистрации пользователя - recordUserConcert
	Обязательные параметры(GET)
		email - Ваш емайл, в случае не корректного email или пустого значения, выдаст ошибку на этот параметр
		
		fullname - Ваше ФИО, в случае  пустого значения, выдаст ошибку на этот параметр
		
		concert- type-integer - принимает целое число.Событие, в случае пустого значения, выдаст ошибку на этот параметр
		Чтобы узнать имя концерта(id) воспользуйтесь параметром getListConcert
		
	Необязательные параметры (GET)
		phone - номер телефона
		
		dob   - дата рождения, формат даты 1994-07-17
		
		gender - пол муж/жен
		
	Данные возвращаются в формате json
     data['success'] == 0 - Означает что возникла ошибка.
     data['messages'] - Сообщение об ошибке если data['success'] == 0;

    возвращается 1 параметр data["success"] == 1 если ошибок не произошло - говорит об успешной регистрации 	 

Пример запроса: ?controller=api/concert&action=recordUserConcert&email=email&phone=phone&concert=12"
2 Метод получения событий - getConcertByLimit 
	Необязательный параметр(GET)
		limit - type integer - принимает целое число какое количество данных необходимо вернуть, в случае если
		 данные будут переданы в качестве text пример : 'afafaf' вернёт 0 значений
